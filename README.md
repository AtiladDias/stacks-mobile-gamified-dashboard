# Stacks Mobile Gamified Dashboard

A mobile-style onchain activity tracker built for the Stacks ecosystem.
This dashboard transforms your wallet activity into XP, levels, streaks,
missions and badges — creating a gamified layer for your Web3 identity.

## Features
- 📱 Mobile-first UI (simulated app experience)
- 🔍 Onchain activity tracking (transactions, active days, protocols)
- 🧩 XP & Level system
- 🔥 Daily & weekly streaks
- 🎯 Missions system (daily quests)
- 🏅 Gamified badges & achievements
- ⚡ API endpoints for XP, streaks, missions, profile
- 🛠 Built with Next.js 14, TypeScript, Wagmi, Stacks.js
